Summary

list of characters Emmet uses and their respective meanings:

>
This character is used to insert RHS element into LHS element so that LHS element wraps RHS element. It is used to create nested structure of html elements. (All example formulas)
*
This character is used to multiply, i.e. duplicate, LHS element by number specified on RHS. (All example formulas)
[]
These brackets are used to specify attribute for element present at it's left hand side. (Example formula 2 to 5)
{}
These brackets are used to specify contents of the html element present on it's LHS. (Example formula 4 & 5)
$
This character is used to add a iterator in the emmet formula. (Example formula 5)
